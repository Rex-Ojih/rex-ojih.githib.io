What I've done so far

Movie Theater Tickets

Languages & Technologies Used

  • C
  • File Handling
  • Basic Data Structures

Description

I created application which makes use of file handling and some of the basic data structures such as linked-lists, queues, stacks and binary search trees. The application was created to allow the user to search movie theaters by zip code, see the available seats in the movie theater, purchase movie tickets and update the available seats, as well as print the customers' receipts. The application had to keep track of and store all the customer transactions as well as the available seats of all the movie theaters. The available seats were written to file which would be interpreted by the application.

Mav Pet Shelter

Languages & Technologies Used

  • OOP
  • C++
  • GTKMM

Description

In this project I used of all the main principals Object-Oriented Programming to create a program for an animal shelter which allows the user to admit and adopt animals to and from the animal shelter. Each animal type was created as a different class which was derrived from an Animal class. Encapsulation, abstraction, inheritance and polymorphism were all used in the creation of this application. I also created a GUI using GTKMM to allow easy interaction between the user and the application.

Animal ID

Languages & Technologies Used

  • Python
  • Tensorflow
  • Java
  • Android Studio

Description

Lead a team project where we used the scrum framework and UML diagrams to assist in creating an Android application which would allow the user to upload an image of an animal and return the species of that animal on screen. In order to do this, I created and trained a convolutional neural network (CNN) using Python and Tensorflow to classify 10 different animal species. The front end of our application was created using Android Studio. In oder to integrate the CNN into the Android application, the CNN was saved as a Tensorflow Lite file. This allowed the Android application to run the CNN on images submitted by the user.
